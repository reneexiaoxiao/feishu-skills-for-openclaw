# 飞书读取文档

## 工具调用

使用 `feishu_fetch_doc` 工具读取文档。

### 参数结构

```json
{
  "document_id": "doxcnXXX",
  "mode": "full"
}
```

### 参数说明

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| document_id | string | ✅ | 文档ID（doxcn开头） |
| mode | string | ❌ | 读取模式（full/summary，默认full） |
| block_id | string | ❌ | 内容块ID（可选） |

## 读取模式

### 模式1：完整读取（默认）

**适用场景**：获取文档全部内容

**用户输入**：
```
读取文档 https://xxx.feishu.cn/docx/doxcnXXX
```

**API 调用**：
```json
{
  "document_id": "doxcnXXX",
  "mode": "full"
}
```

### 模式2：生成摘要

**适用场景**：快速了解文档内容

**用户输入**：
```
给我看看这个文档的摘要：doxcnXXX
```

**API 调用**：
```json
{
  "document_id": "doxcnXXX",
  "mode": "summary"
}
```

### 模式3：读取指定块

**适用场景**：只读取特定部分

**用户输入**：
```
读取文档 doxcnXXX 的第3段
```

**API 调用**：
```json
{
  "document_id": "doxcnXXX",
  "mode": "block",
  "block_id": "block_xxx"
}
```

## 执行流程

### 1. 提取文档ID

从用户输入中提取文档ID：

**方式1：从URL提取**
```
用户：读取 https://xxx.feishu.cn/docx/doxcnXXX
→ 提取：doxcnXXX
```

**方式2：直接提供**
```
用户：读取文档 doxcnXXX
→ 直接使用：doxcnXXX
```

**方式3：从文档名称搜索**
```
用户：读取"会议纪要"文档
→ 搜索文档获取ID
```

### 2. 确定读取模式

根据用户输入判断：

| 关键词 | 模式 |
|--------|------|
| 读取、获取、查看 | full（完整） |
| 摘要、概要、总结 | summary（摘要） |
| 第几段、某部分 | block（指定块） |

### 3. 调用读取工具

### 4. 处理返回内容

根据内容大小和用户需求，决定返回方式：
- 小文档：完整返回
- 大文档：分页或摘要返回
- 需要分析：提供结构化摘要

## 常见场景

### 场景1：读取完整文档

**用户输入**：
```
读取文档 doxcnXXX 的内容
```

**执行**：
1. 调用读取工具
2. 返回完整内容

**返回示例**：
```
# 项目计划

## 目标
- 完成API开发
- 进行测试

## 进度
当前进度：80%
```

### 场景2：读取文档摘要

**用户输入**：
```
给我这个文档的摘要：https://xxx.feishu.cn/docx/doxcnXXX
```

**执行**：
1. 读取文档
2. 生成摘要

**返回示例**：
```
文档摘要：
- 标题：项目计划
- 主要内容：API开发和测试计划
- 当前进度：80%
- 关键节点：已完成设计，正在开发
```

### 场景3：读取特定部分

**用户输入**：
```
读取文档 doxcnXXX 的"进度"章节
```

**执行**：
1. 读取文档结构
2. 定位"进度"章节
3. 返回该章节内容

### 场景4：大文档处理

**用户输入**：
```
读取文档内容（假设文档很长）
```

**执行**：
1. 检测文档大小
2. 如果过大，提供选项：
   - 返回完整内容（可能很长）
   - 返回摘要
   - 分页返回

**处理**：
```
文档较长（5000字），请问需要：
1. 完整内容
2. 摘要
3. 分页查看（每页1000字）
```

## 错误处理

### 错误1：文档ID无效

**处理**：
```
"文档ID格式错误或文档不存在
正确格式：doxcnXXX（从文档URL获取）"
```

### 错误2：无权限访问

**错误码**：99991403

**处理**：
```
"无权限访问此文档
可能原因：
1. 文档未分享给你
2. 需要登录
3. 文档已删除"
```

### 错误3：文档不存在

**错误码**：99991410

**处理**：
```
"文档不存在或已删除
请检查文档ID是否正确"
```

### 错误4：从名称搜索失败

**处理**：
```
"找不到名为"XXX"的文档
请提供：
1. 完整文档名称
2. 或直接提供文档ID"
```

## 注意事项

1. **文档ID格式**：必须是 doxcn 开头的完整 ID
2. **访问权限**：确保有文档的访问权限
3. **内容大小**：大文档考虑分页或摘要
4. **隐私保护**：不分享敏感文档内容
5. **格式保留**：尽量保持原文档格式

## 返回内容处理

### 小文档（< 2000字）

直接返回完整内容。

### 中等文档（2000-5000字）

返回完整内容，提供分页选项。

### 大文档（> 5000字）

默认返回摘要，提供完整内容选项。

**处理示例**：
```
文档较长（8000字），已生成摘要：
[摘要内容]

需要查看完整内容吗？回复"完整内容"获取全部。
```
