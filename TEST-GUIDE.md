## 故障排查

### 技能未被调用

**症状**：AI 只是解释，不执行操作

**可能原因**：
1. Skill 未正确加载
2. 触发条件不明确
3. 对话意图识别错误

**解决方法**：
1. 检查 Skill 是否在 `~/.openclaw/skills/` 目录
2. 重启 OpenClaw
3. 使用明确的动作词（发送、创建、更新）

### 参数错误

**症状**：API 调用失败

**可能原因**：
1. ID 格式错误
2. 必填参数缺失
3. 参数类型不匹配

**解决方法**：
1. 检查 ID 格式（oc_、ou_、doxcn、app_、tbl_、rec_）
2. 提供完整参数
3. 确认参数类型正确

### 权限错误

**症状**：无权限访问

**可能原因**：
1. 机器人未加入群聊
2. 无文档访问权限
3. 无表格编辑权限

**解决方法**：
1. 将机器人加入群聊
2. 获取文档访问权限
3. 获取表格编辑权限
