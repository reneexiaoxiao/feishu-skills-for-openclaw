```yaml
  (1) ...
  (2) ...
  (3) ...
  (4) ...

  - ...
  - ...

  **参数说明**：
  - ...
  - ...
```

### v2.0 格式（精简）
```yaml
---
name: feishu-send-card
description: |
  向飞书群聊或私聊发送交互式卡片。

  **使用此技能**：用户说"发卡片"、"卡片消息"时。

  **调用工具**：feishu_im_user_message (来自 @larksuite/openclaw-lark)

  **关键参数**：
  - receive_id: 群聊ID (oc_xxx)
  - msg_type: "interactive"
  - content: 卡片JSON
```
