## [1.0.0] - 2026-03-11

### 新增
- 🎉 首次发布飞书技能系列
- 💬 即时通讯技能（4个）
  - send-message - 发送文本消息
  - send-card - 发送卡片
  - send-image - 发送图片
  - mention-user - @提及用户
- 📄 文档操作技能（3个）
  - create-doc - 创建文档
  - update-doc - 更新文档
  - fetch-doc - 读取文档
- 📊 数据管理技能（1个）
  - bitable - 多维表格操作
- 📅 日程管理技能（1个）
  - calendar - 日程管理

### 特性
- ✨ 每个技能独立，职责单一
- 🚀 识别意图立即执行，不教学
- 🔧 易于排查和维护
- 🧦 可组合使用完成复杂任务
- 📚 完整的文档和测试指南

### 工具
- 🔧 一键安装脚本
- 📖 完整的 README 和测试指南
